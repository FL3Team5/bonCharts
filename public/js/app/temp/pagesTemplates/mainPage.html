Hello, world!

This is a template for a simple marketing or informational website. It includes a large callout called a jumbotron and three supporting pieces of content. Use it as a starting point to create something more unique.

Bar

Bar chart is a chart when bars of a different graphs are placed next to each another, and not stacked one on another, so basically it’s the most simple bar chart with two graphs in it.

Line

All our serial charts (and actually XY chart too) can be zoomed-in, scrolled and panned. All you have to do is add a scrollbar to the chart. This is one of the most important features of charts, especially when dealing with large data sets.

Area

Technically, area graph is just a regular line graph with fill opacity set to some bigger than 0 value. This means you can easily switch from area to line only by modifying this property. You can also define which of the graphs are hidden by default.

Pie

Our pie chart uses intelligent way to arrange it’s labels so that they would not overlap. You can try adding even more labels by pressing EDIT button and adding more items to the data provider – the labels will be arranged so that they wouldn’t overlap.

Donut

Donut or Doughnut chart is just a simple pie chart with a hole inside. You can define hole radius to any size you need, both in percent or pixels.

Scatter

A scatter plot, scatterplot, or scattergraph is a type of mathematical diagram using Cartesian coordinates to display values for typically two variables for a set of data.
